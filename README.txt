Aplikasi ini dibuat untuk melatih teman-teman yang tidak fasih komputer agar bisa memanfaatkan komputer untuk mengikuti ujian berbasis CAT.
Aplikasi ini tidak berafiliasi dengan BKN, dibuat hanya berupa single file static HTML dengan Javascript dengan soal dummy.
Semoga dengan adanya aplikasi ini, teman-teman yang tidak fasih menggunakan komputer bisa lebih lancar ujian menggunakan CAT.

Cara penggunaan:
1) Pastikan Google Chrome / browser lain terinstall
2) Extract ZIP ini di suatu folder
3) Klik SIMULASI.html agar dibuka di browser

Catatan:
- HARAP DIJALANKAN MENGGUNAKAN PC / LAPTOP, KEMUDIAN FULL SCREEN (F11) UNTUK BEST EXPERIENCE
- Hanya di test di Google Chrome terakhir dan di PC (tidak pernah dicoba di smartphone).
- Mungkin saja ada bug
- No warranty
- Bebas dimodif jika diinginkan
